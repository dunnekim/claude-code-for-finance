# Part 6: DART 실습 — 1개사 재무 분석

> **형태:** 실습 (40분)
>
> **목표:** OpenDart API로 재무 데이터를 가져오고, 기본 재무 분석을 수행

---

## 두 가지 경로

이 실습은 **API 경로**와 **샘플 데이터 경로** 두 가지로 진행할 수 있습니다. 어느 쪽이든 학습 효과는 동일합니다.

| 경로 | 상황 | 방법 |
|------|------|------|
| **A. API 경로** | API 연결 정상 | OpenDart API로 실시간 데이터 조회 |
| **B. 샘플 경로** | API 오류, 네트워크 문제 | `실습데이터/` 폴더의 CSV 파일 사용 |

Part 5에서 API 연결 테스트를 통과했으면 A 경로로, 실패했으면 B 경로로 진행하세요.

## 6-1: 기업 개황 조회 (10분)

### A 경로 — API 사용

```
OpenDart API를 사용해서 우리금융지주의 기업 개황 정보를 가져와줘.

참고:
- API 키는 환경변수 OPENDART_API_KEY 사용
- 우리금융지주 종목코드: 316140
- OpenDart는 종목코드가 아닌 고유번호(corp_code)를 사용하므로,
  먼저 기업 고유번호를 조회해야 할 수 있어
```

### B 경로 — 샘플 데이터 사용

```
실습데이터/우리금융지주_연결재무제표.csv 파일을 읽고,
어떤 기업의 어떤 기간 데이터인지 요약해줘.
```

> **API 경로에서 에러가 나면?** 당황하지 마세요.
> "에러가 났어. 고쳐줘."라고 하면 Claude가 스스로 수정합니다.
> 2~3번 시도해도 안 되면 B 경로로 전환하세요. 시간을 낭비하지 않는 것이 중요합니다.

## 6-2: 재무제표 데이터 추출 (15분)

### A 경로

```
우리금융지주의 최근 3개년(2022, 2023, 2024) 연결 재무제표에서
아래 항목을 추출해줘:

손익계산서: 영업수익, 영업이익, 당기순이익
재무상태표: 자산총계, 부채총계, 자본총계

참고:
- 우리금융지주는 금융지주이므로, 매출액이 아닌 "영업수익" 항목을 사용해
- API 키는 환경변수 OPENDART_API_KEY를 사용
- 결과를 표 형태로 보여주고, analysis/우리금융지주_재무데이터.md에도 저장해줘
```

### B 경로

```
실습데이터/우리금융지주_연결재무제표.csv를 읽고,
연도별로 영업수익, 영업이익, 당기순이익, 자산총계, 부채총계, 자본총계를
표 형태로 정리해줘.
analysis/우리금융지주_재무데이터.md에 저장해줘.
```

## 6-3: 재무 분석 (15분)

여기서부터는 A/B 경로 모두 동일합니다.

### 미션 3: 재무비율 분석

```
방금 정리한 우리금융지주 재무 데이터를 기반으로:

1. 성장성: 영업수익 성장률(YoY), 영업이익 성장률
2. 수익성: 영업이익률, 순이익률
3. 안정성: 부채비율, 자기자본비율
4. 효율성 지표: ROE, ROA
5. 종합 코멘트: 3개년 추이 해석, 주요 변동 요인 추정

결과를 analysis/우리금융지주_재무분석.md에 저장해줘.
```

### 미션 4: 결과 검토

analysis/ 폴더에서 생성된 파일을 열어 확인합니다.

**직접 검증하세요:**
- 숫자가 합리적인가? 백만원 단위가 적용되었는가?
- 비율 계산이 정확한가? (특히 ROE = 당기순이익 / 자본총계)
- 영업수익이 매출액으로 잘못 표시되지는 않았는가?
- 코멘트가 논리적인가?

> **AI 산출물은 항상 검증이 필요합니다.**
> "ROE 계산 과정을 보여줘"처럼 검산을 요청하세요.
> Part 2에서 말한 "PM이 판단에 집중"하려면, AI의 계산은 신뢰하되 검증하는 습관이 필요합니다.

## 금융업 재무 분석의 특수성

이번 실습에서 우리가 사용한 항목은 일반 제조업과 다음과 같이 다릅니다:

| 일반 제조업 | 금융업 | 비고 |
|-------------|--------|------|
| 매출액 | 영업수익 | 이자수익 + 수수료수익 + 기타 |
| 매출총이익 | (없음 또는 NIM) | 금융업은 매출원가 개념이 다름 |
| ROE / ROA | ROE / ROA | 동일 |

CLAUDE.md에 추가한 "금융업 매출 = 영업수익" 규칙이 이 차이를 자동으로 처리해 줍니다.
