# Part 4: 환경 확인 + Claude Code 설치

> **형태:** 실습 (15분)
>
> **목표:** 사전 세팅 확인, VS Code 익스텐션 설치, 인증 완료

---

## Step 1: 사전 세팅 확인 (5분)

VS Code를 실행하고, 내장 터미널을 엽니다 (`` Ctrl+` ``).

아래 명령어를 하나씩 실행합니다.

```
git --version
python --version
echo %OPENDART_API_KEY%
python -c "import requests; print('OK')"
python -c "import bs4; print('OK')"
```

5개 모두 정상 출력되면 통과입니다.

> **하나라도 실패하면** [Part 1 사전 세팅](01_사전세팅.md)을 다시 확인하거나 강의자에게 알려주세요.

---

## Step 2: 실습 폴더 열기 (2분)

1. VS Code에서 `File → Open Folder`
2. ZIP을 압축 해제한 위치의 `claude-code-for-finance/실습환경/` 폴더 선택
3. 왼쪽 파일 탐색기에서 구조 확인

---

## Step 3: Claude Code Extension 설치 (5분)

1. `Ctrl+Shift+X` → 검색창에 **"Claude Code"** 입력
2. **Anthropic**이 게시한 Extension → **Install**
3. 왼쪽 사이드바에서 **Claude Code 아이콘** (스파크 모양) 클릭
4. **"Claude.ai subscription"** 버튼 → 브라우저에서 로그인 → 승인

채팅 패널에 입력창이 활성화되면 인증 완료입니다.

---

## Step 4: 첫 대화 (3분)

```
안녕! 현재 프로젝트 폴더의 구조를 보여줘.
```

Claude가 폴더 구조를 읽어서 보여주면 정상입니다.

권한 승인 팝업이 뜨면 내용을 확인하고 **Yes**를 누르세요.

---

## 트러블슈팅

| 증상 | 해결 |
|------|------|
| Extension 검색 안 됨 | VS Code 업데이트 |
| 인증 후 연결 안 됨 | Free 플랜 → Pro 이상 필요 |
| `python` 명령어 실패 | CMD/터미널 닫고 다시 열기 (PATH 적용), 또는 [Part 1](01_사전세팅.md) winget 재실행 |
| `import requests` 실패 | `pip install requests beautifulsoup4` 다시 실행 |
| 환경변수 안 읽힘 | VS Code 완전히 종료 후 재실행 (환경변수는 프로세스 시작 시 한 번만 로딩됨) |
