## 7-1: 추가 4개사 데이터 수집 (15분)

Part 6에서 우리금융지주(316140) 한 곳을 분석했습니다. 이제 같은 작업을 다른 금융 4개사에 반복합니다.

**대상 4개사:**

| 회사 | 종목코드 | 업종 |
|------|---------|------|
| 동양생명 | 082640 | 생명보험 |
| 롯데손해보험 | 000400 | 손해보험 |
| 카카오뱅크 | 323410 | 인터넷은행 |
| 키움증권 | 039490 | 증권 |

### A 경로 — API 사용

```
동양생명(082640), 롯데손해보험(000400), 카카오뱅크(323410), 키움증권(039490)도
우리금융지주와 동일한 방식으로 최근 3개년 연결 재무제표를 가져와줘.

동일 항목: 영업수익, 영업이익, 당기순이익, 자산총계, 부채총계, 자본총계
(모두 금융업이므로 매출액이 아닌 영업수익 사용)

각각 analysis/ 폴더에 저장:
- analysis/동양생명_재무데이터.md
- analysis/롯데손해보험_재무데이터.md
- analysis/카카오뱅크_재무데이터.md
- analysis/키움증권_재무데이터.md
```

### B 경로 — 샘플 데이터 사용

```
실습데이터/ 폴더에 있는 동양생명, 롯데손해보험, 카카오뱅크, 키움증권 CSV 파일을 읽고,
우리금융지주와 동일한 형식으로 정리해줘.
각각 analysis/ 폴더에 저장.
```

### 개별 분석

```
동양생명, 롯데손해보험, 카카오뱅크, 키움증권 4개사도 우리금융지주와 동일하게
재무비율 분석을 수행해줘.

영업수익 성장률, 영업이익률, 순이익률, 부채비율, ROE, ROA 포함.
각각 analysis/ 폴더에 저장.
```

## 7-2: 5개사 비교 분석 보고서 (15분)

### 미션: 5개사 종합 비교

```
analysis/ 폴더의 우리금융지주, 동양생명, 롯데손해보험, 카카오뱅크, 키움증권
분석 결과를 종합해서 비교 분석 보고서를 만들어줘.

구조:
1. 요약 (3줄 이내)
2. 주요 지표 비교 테이블 (5개사 × 3개년)
3. 업종별 특성 요약 (지주·생보·손보·인터넷은행·증권)
4. 수익성 비교 — 영업이익률 추이
5. 안정성 비교 — 자기자본비율 (금융업은 자본 적정성이 핵심)
6. 효율성 비교 — ROE, ROA
7. 종합 의견

결과를 analysis/금융5사_비교분석.md에 저장해줘.
```

## 여기서 멈추고 생각해 봅시다

Part 6~7에서 당신이 한 일:

1. 우리금융지주 재무 데이터 가져오기
2. 재무비율 분석
3. **동양생명도 같은 방식으로** ← 반복
4. **롯데손해보험도 같은 방식으로** ← 또 반복
5. **카카오뱅크도 같은 방식으로** ← 또 반복
6. **키움증권도 같은 방식으로** ← 또또 반복
7. 5개사 비교

**"종목코드를 넣으면 → 재무제표를 가져오고 → 분석하고 → 저장한다"**

이 패턴이 5번 반복됩니다. 매번 긴 프롬프트를 입력하고 있습니다. 한 회사당 약 5분, 5개사면 25분. 만약 회사가 10개라면 50분, 20개면 100분.

**다음 파트에서 이 패턴을 한 줄 명령어로 만듭니다.** 25분짜리 작업이 5번의 한 줄 명령어가 됩니다.
