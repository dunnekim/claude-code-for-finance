# Claude Code for Finance — 설치부터 스킬 구축까지

> 3시간 후, 한 작업이 한 줄이 됩니다. 그리고 같은 패턴으로 다른 작업도 만들 수 있게 됩니다.

회계법인 회계사를 위한 Claude Code 반나절 워크숍.

## 이 워크숍은 왜 존재하는가

우리는 소규모 팀제로 일합니다. 스탭이 만든 산출물의 품질은 들쭉날쭉하고, PM이 리뷰에서 놓치는 것은 반복되며, 시간이라는 물리적 병목 앞에서 모든 팀원이 고통받고 있습니다.

이 워크숍은 "AI 사용법"을 가르치는 자리가 아닙니다. **DART 분석이라는 한 작업을 자동화하면서, 그 자동화 패턴을 익혀 다른 반복 업무에도 적용할 수 있게 만드는** 자리입니다. 거창한 "팀 전체 AI 인프라"를 약속하지 않습니다. 한 작업의 가치를 손으로 검증하고, 같은 패턴으로 다음 작업을 30분 안에 만들 수 있다는 것을 체험합니다.

> - **대안:** [7-Zip](https://www.7-zip.org) — 글로벌 표준 무료 도구
> - **가장 안전한 방법:** GitHub에서 직접 보거나 `git clone`으로 받기 (압축 해제 불필요)

## 사전 준비 체크리스트

| 항목 | 명령어/링크 |
|------|------------|
| VS Code 설치 | code.visualstudio.com |
| Git + Python 3.12 (winget 한 줄) | `winget install Git.Git && winget install Python.Python.3.12` |
| Python 패키지 | `pip install requests beautifulsoup4` |
| Claude Pro + VS Code Extension | claude.ai → Pro · VS Code Marketplace에서 "Claude Code" |
| OpenDart API Key + 환경변수 ⭐ | `setx OPENDART_API_KEY "발급받은_키"` |
| 실습 폴더 다운로드 | 이 저장소 git clone 또는 ZIP |

자세한 단계는 [Part 0: 사전 세팅 가이드](01_사전세팅.md) 참고.

## 실습 폴더 구조

```
실습환경/
├── 실습데이터/                 ← 금융 5개사 학습용 CSV (백업)
├── scripts/                   ← Python 스크립트 (DART, 네이버금융)
├── analysis/                  ← 분석 결과 저장 위치
├── .claude/
│   ├── skills/예시분석/        ← 사전 배치된 예시 Skill
│   └── agents/                ← 데모용 Sub-agent (데이터수집가, 재무분석가)
└── CLAUDE.md                  ← 보안 규칙 (Part 4에서 프로젝트 규칙 추가)
```
