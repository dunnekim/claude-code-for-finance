## 데이터 항목

각 회사 3개년(2022~2024) 연결재무제표:
- 손익계산서: 영업수익, 영업이익, 당기순이익
- 재무상태표: 자산총계, 부채총계, 자본총계

## ⚠ 중요 — 학습용 샘플 데이터

**이 CSV는 워크숍 학습 목적의 합리적 추정치이며, 실제 DART 공시 데이터와 차이가 있을 수 있습니다.**

API가 정상 작동하는 환경에서는 OpenDart API로 실시간 데이터를 가져오는 것을 권장합니다 (Part 5 A 경로). 이 CSV는 API 실패 시 백업 또는 오프라인 학습용입니다.

워크숍에서 만들어지는 분석 결과는 실제 투자 판단에 사용해서는 안 됩니다.

## 금융업 항목명 주의

전자·제조업과 달리 금융업은 **영업수익**(이자수익 + 수수료수익 + 기타)이 매출에 해당합니다. 일반적인 "매출액" 항목은 사용하지 않습니다. 자세한 내용은 [실습환경/CLAUDE.md](../CLAUDE.md)의 프로젝트 규칙을 참고하세요.
