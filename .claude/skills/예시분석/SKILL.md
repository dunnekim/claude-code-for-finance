$ARGUMENTS 파일을 분석해주세요.

## 분석 절차

1. CSV 파일을 읽고 데이터 구조를 파악합니다
2. 연도별 주요 항목(영업수익, 영업이익, 당기순이익, 자산총계, 부채총계, 자본총계)을 표로 정리합니다
   - 금융업(은행·보험·증권·금융지주)의 매출은 "영업수익"으로 표기됩니다
3. 아래 재무비율을 계산합니다:
   - 영업수익 성장률 (YoY)
   - 영업이익률 (영업이익 / 영업수익)
   - 자기자본비율 (자본총계 / 자산총계)
   - ROE (당기순이익 / 자본총계)
   - ROA (당기순이익 / 자산총계)
4. 3개년 추이에 대한 간단한 코멘트를 작성합니다
5. 결과를 analysis/ 폴더에 마크다운으로 저장합니다

## 출력 규칙

- 금액 단위: 백만원 (소수점 이하 반올림)
- 비율: 소수점 첫째자리까지 (예: 14.3%)
- 파일명: analysis/[기업명]_예시분석.md
